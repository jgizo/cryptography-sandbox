Affine Cipher

The Affine cipher is a mono-alphabetic substitution that combines a multiplicative key and an additive key: 

Encryption :  C = (a · P + b) mod 26
Decryption :  P = a⁻¹ · (C − b) mod 26

Two integers form the key pair (a, b).  a must be coprime with 26 so that its modular inverse a⁻¹ exists.

Example we’ll follow:
plaintext = AFFINE,  a = 5,  b = 8

Step 2 — Decryption table

a = 5 ⇒ a⁻¹ = 21 because 5·21 ≡ 1 (mod 26).

We recover the original plaintext AFFINE.
